# Music Store Data Analysis using SQL

## About the Project
In this project, I analyzed a Music Store database using SQL.  
The goal was to understand the data and find useful information from it.

---

## What I Did
- Analyzed customers and invoices
- Found top spending customers
- Identified countries and cities with the highest revenue
- Found the most popular music genres by country
- Analyzed artists and track sales

---

## SQL Concepts Used
- JOINs
- GROUP BY and ORDER BY
- Aggregate functions (SUM, COUNT, AVG)
- CTEs
- Window functions

---

## Questions Answered
1.Who is the senior most employee based on job title?

2.Which countries have the most invoices?

3.What are the top 3 invoice totals?

4.Which city generates the highest total revenue?

5.Who is the best customer (highest spending)?

6.Which customers listen to Rock music?

7.Which artists have written the most Rock music?

8.Which tracks are longer than the average song length?

9.How much did customers spend on the best-selling artist?

10.What is the most popular music genre in each country?

11.Who is the top customer by spending in each country??

---

## What I Learned
- How to write SQL queries to solve real problems
- How to work with relational databases
- How to analyze data and find insights

---

## Dataset
This project uses the Music Store (Chinook) database.
